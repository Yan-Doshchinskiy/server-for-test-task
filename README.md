# Server runs on http://localhost:5000

Сервер для https запросов работает на порту 5000 на локале.
На Heroku порт присваивается автоматически, поэтому напрямую к порту обращаться не нужно.
Порт 5000 привязан к адресу через server.listen(port), поэтому все запросы нужно отправлять без указания порта.

Сервер для https запросов: https://chat-https-server.herokuapp.com

Для websocket запросов запущен другой сервер по адресу: ws://chat-ws-server.herokuapp.com

Для хеширования паролей используется bcript.
Токен для авторизации генерируется через jwt.
База данных mongoose(mongoDB).
